Le tableau comporte une ligne pour chaque aspect essentiel des compétences analytiques : 

* Curiosité : désir d’en savoir plus sur quelque chose, en posant les bonnes questions
* Comprendre le contexte : comprendre où l’information s’inscrit dans la « vue d’ensemble »
* Avoir une optique technique : diviser les grandes choses en plus petites étapes
* Design de données : penser à la manière d’organiser les données et les informations
* Stratégie de données : penser aux personnes, aux processus et aux outils utilisés en analyse de données 


Vous mettrez une croix X dans la colonne qui, selon vous, décrit le mieux votre niveau actuel pour chaque aspect. Les trois notations sont les suivantes :

* Force : c’est un domaine qui, selon vous, est l’une de vos forces.
* Développement : vous avez une certaine expérience dans ce domaine, mais il y a encore une marge de progression importante.
* Émergence : c’est nouveau pour vous, et vous acquérez de l’expérience dans ce domaine à partir de ce cours. 

Ensuite, mettez à jour la colonne Commentaires/Plans/Objectifs avec une note rapide sur les raisons pour lesquelles vous avez choisi ces notations.
